## 📖 About

**3D YouTube Downloader Batch 2.14.2** is a lightweight Windows utility designed to download YouTube videos and audio in batch mode. It supports high-resolution downloads, including 3D and 4K content, and offers a simple interface for queuing multiple URLs at once. This tool is intended for personal, offline use of legally acquired content.
